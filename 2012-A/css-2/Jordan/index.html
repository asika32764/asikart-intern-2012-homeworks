蘇拉恐二次登陸　影響台灣時間增長8小時

目前蘇拉以每小時12轉15公里速度，向北北東轉西北進行，儘管各地風雨仍強，但蘇拉登陸後，有減弱趨勢。 根據最新消息，蘇拉強度已略減為中颱下限，但台灣各地包含綠島、蘭嶼、澎湖及馬祖地區，仍應嚴加戒備並防強風豪雨。 由於蘇拉慢步前進，預計影響台灣的時間可能增長6-8個鐘頭。而其帶來的雨量也相當驚人。宜蘭山區雨量已上修到2000毫米，花蓮山區則為1100毫米。 另外，蘇拉帶來的風勢也有改變趨勢，由東北轉吹西北，屆時桃竹苗等地可能因此出現較大雨勢，而就連背風的台東大武也受影響，在清晨5時已出現36.8度的焚風。至於最大陣風出現在蘇澳，有14級，也掀起10-11米的浪高，基隆最大陣風則是12級。

超殺 莊智淵乒進4強寫歷史

「桌球小子」莊智淵昨在倫敦奧運男子單打八強賽中登場，現場聚光燈打在場館中央的桌球台上，莊智淵儼然就是主角，並順利以四比○輕取羅馬尼亞選手克里森，寫下我國奧運男桌史上最佳紀錄。台北時間今晚六點，將在四強準決賽出戰大會第二種子中國隊王皓。小莊說：「我很期待，如果這次再輸，就不是技術問題了！」克里森何以這麼說呢？因為他手持的球拍就是莊智淵參加京奧用的，他四度參加奧運，第一次晉級八強，也締造個人最佳紀錄。 莊智淵滿臉鬍渣、眼神酷狠，倫奧志工封他為「桌球機器」，隱喻他接發球像機器般精準，過去打球毛躁的「火爆小子」，現在已蛻變得精練成熟。下一場迎戰在雅典奧運八強賽擊敗他的大陸隊王皓，莊智淵滿懷信心。 男桌八強賽開鑼，現場開始有主持人帶動氣氛，屋樑上六台懸掛式螢幕播放歷來奧運桌球菁華片段，並用電腦動畫介紹桌球各種技巧打法，當莊智淵和克里森被介紹出場時，氣氛已凝聚，就像等著一場精采的秀上演。

「南部沒下雨，菜到哪去?」／菜價漲太兇 主婦買到一肚子火

颱風來襲，菜價應聲上揚。宜蘭三星蔥每公斤平均批發價突破一百五十元，是半個多月前的三倍價。彰化縣溪湖果菜市場一日部分蔬菜漲幅高過平均漲幅，青蔥漲四成，小黃瓜也有三成漲幅，雲林部分，小白菜、小辣椒等漲幅都在二倍以上。小白菜、小辣椒漲幅逾2倍 「老闆，到底哪樣菜便宜？」「只要是菜，統統不便宜！」傍晚台北的黃昏市場，人潮並未因為狂風暴雨減少，但瞬間飆漲的菜價卻貴到讓人買不下去，「芹菜一斤一百三十元、高麗菜挑最小顆的還要上百元…！」主婦廖太太抱怨：「老闆們都說是北部的菜、搶收的菜，中南部沒下雨的菜哪兒去了？根本就是藉機發颱風財！」 宜蘭因下起超大豪雨，三星鄉蔥農擔心青蔥受災，冒雨搶收。三星蔥半個多月前批發價約五十到六十元間，七月十五日過後，中南部青蔥產地因雨受創，出貨量減少，三星蔥價水漲船高，這幾天又有颱風效應，昨天的批發均價達到一百五十二元，全鄉供應量超過二十噸。
