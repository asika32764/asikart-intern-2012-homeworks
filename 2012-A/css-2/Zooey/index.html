Dazzling Cafe

Dazzling Cafe Deluxe結合上東城的奢華與香榭麗舍的精緻文化，運用上等食材，推出頂級蜜糖吐司、英式下午茶high tea，當然也提供義大利麵、三明治、開胃菜、湯品等輕食。Dazzling Cafe Deluxe的嚴選茶品同樣令人驚艷，除了法國國寶茶葉第一品牌Mariage Frères(瑪黑茶)，還有同樣來自法國的百年品牌HEDIARD、水果口味的希臘Frappe系列特飲等，呈現風格不同的蜜糖新主張。
